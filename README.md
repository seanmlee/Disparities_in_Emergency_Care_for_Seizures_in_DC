
## Overview

- This repository includes reproducible code for our research manuscript titled "<em>The Role of Demographic Factors of Individuals with Seizures in Emergency Care Delivery</em>". The manuscript is currently in submission/in review.
- We will provide data, the full citation, abstract, and visual abstract upon publication
<br>  

## How to use this repository

- Clone the repository and open the `./Disparities_in_Emergency_Care_for_Seizures_in_DC.Rproj` R Project file
- This repository uses `renv` dependency management; all software, package, and dependency version information is listed in the `./renv.lock` file and will automatically configure once prompted
- This repository also uses environment variables (via a .Renviron file) to define file paths, ensuring compatibility across Mac and Windows operating systems
- All scripts are in the `./scripts` directory
- Execute scripts in their numbered order to format data, fit models, and generate tables and figures
- Please direct any questions to smlee@gwu.edu
